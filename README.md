# **React TODO**

This is a Simple TODO list application built using JavaScript's React library. It provides a simple user interface and the core functionality of a TODO list.

No back-end is developed. All storage is done through the browser's local storage.

# **Features**
## **Curent Features**
* Add tasks: Create and add new todo tasks to your list.
* Complete tasks: Mark tasks as completed.
* Delete tasks: Remove completed or unwanted tasks.

## **Future Development**
* Update task: Edit a todo task already on the list.
* Animations: Add animations for a more professional feel.

# **Technologies Used**

* React
* HTML
* CSS
